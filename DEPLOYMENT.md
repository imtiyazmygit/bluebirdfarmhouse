## Quick Preview

You can view the website by:

1. **Cloning the repository**:
   ```bash
   git clone https://github.com/imtiyazmygit/bluebirdfarmhouse.git
   cd bluebirdfarmhouse
   ```

2. **Opening locally**: Simply open `index.html` in your web browser

3. **Using a local server** (recommended):
   ```bash
   python3 -m http.server 8000
   ```
   Then visit: http://localhost:8000

## Deploying to GitHub Pages

1. Go to repository Settings
2. Navigate to "Pages" section
3. Select branch: `main` (or current branch)
4. Select folder: `/ (root)`
5. Click "Save"

Your site will be live at: `https://imtiyazmygit.github.io/bluebirdfarmhouse/`
